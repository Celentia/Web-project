Урок 3 
Создать конструкторы для товаров интренет магазина.
1.СОздать конструктор для мобильных телефонов.
2.СОздать конструктор для ноутбуков.
3.СОздать конструктор для планшетов.

Придумать список свойств(должны быть - иконка, 3 обычных типа, 2 - вложенных объекта,
!обязательно должно быть свойство, с одинаковым именем, которое идентифицирует телефон это
или что-то другое... например(class: 'phone', class:'tablet') и т.д.)

Товары  записываются в общий массив объектов - goods.


Помимо стандартных методов, которые одинаковые для каждой группы товаров
(toString, valueOf, compareTo, equals) 
есть ещё 1 метод
(добавить в корзину)
Который есть в таблице вывода всех товаров в виде кнопки.

По нажатию на кнопку - элемент по соответсвующему индексу в массиве товаров пушится(push)
в массив cart.
Внизу таблицы вывода всех товаров есть кнопка которая отобразит элементы массива cart.


Не пугайтесь - всё предельно просто. Примеры у Вас есть в архивах с уроков.
Обратите внимание, что в обработчик события нажатия по кнопке "добавить в корзину"
должен передаваться параметр - индекс элемента, чтобы было понятно что пушить.
